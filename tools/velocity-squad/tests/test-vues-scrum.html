🧪 Tests - Vues Multiples Scrum

📋 Instructions de Test

Suivez ces étapes pour valider les nouvelles fonctionnalités des vues Scrum :

  1. Ouvrir l'application principale (index.html)
  2. S'assurer d'être en mode Scrum
  3. Avoir au moins 3 sprints avec des données
  4. Vérifier chaque test ci-dessous

📊 Test 1 : Affichage du Sélecteur de Vues

1.1 - Sélecteur visible en mode Scrum

Le sélecteur avec 3 boutons emoji doit être visible en haut du graphique.

À TESTER Vérifier la présence des boutons 📊 📉 📈

1.2 - Sélecteur caché en mode Kanban

Passer en mode Kanban : le sélecteur doit disparaître.

À TESTER Changer le mode et vérifier

1.3 - Sélecteur caché sans sprints

Supprimer tous les sprints : le sélecteur doit disparaître.

À TESTER Vérifier avec 0 sprint
